Boggle, that game with the cubes!

Project Description

I developed my implementation of web-based Boggle for my final web lab project, but also for my younger self who loved Boggle. I wanted to make a more interactive version of the game I would play with just a pen and paper, but still have the same feel as the original game.

Duration

Screnshot of the game

Building Boggle

My Boggle game is built with Javascript and HTML/CSS. I modeled the cubes after my own Boggle game's cubes, and I implemented the scoring and word-building systems according to the original Boggle rules. I made a 3-minute timer which can be paused and refreshed, but if it is paused the cubes are hidden (so that you don't cheat!). When a word is created, it is first checked against a dictionary, then if it is found in the dictionary, the word is checked against already played words. If the word has been played already or was not in the dictionary it is given a different appearance, and it is not scored. If it is in the dictionary, it is added to the score, which is updated accordingly. At the end of the game a score screen overlays the game screen and the player's score and a randomized message appear alongside a button that resets the board for a new game.
